DESCRIPTION

A mining pool for Mineable Punks NFT

The mining pool is based on contribution from a minor nonce found. A score is tallied for submitted nonces. When a valid mpunk nonce is found the score of the highest address is picked, that address score is set to zero, and the pool starts mining that address.

Nonces found are printed to here. To find the latest valid nonce see latest.

INSTRUCTIONS

LINUX

Download pre-built mminer4 from here

Make it executible. 

chmod +x mminer4

Get dependencies. You can install with sh dependencies.txt

Run with 

./mminer4 -a 'your_eth_address'
For multiple gpu support you can use -x 1 flag where number is the gpu ordinal number

I run mine like ./mminer4 -a 0xE8946EC499a839c72E60bA7d437E28cd73a3f487 > out.log &

If an address is passed in that does not begin with 0x it will be ignored. If no address is passed in (no flag) then it will use the default address.

MINTING

For now have to monitor the score to know when you are mined for and then the nonces. The end of the file contains latest nonce found. (todo: clean up nonce file)

Go to the contract page. Check to see if someone has minted after your nonce is found (nonces are based on last minted assets). The current timezone in the nonces page is UTC. You can translate that using https://www.worldtimebuddy.com/

Convert your nonce into integer using an online converter.

Using a wallet to 'connect to web3' go to the write contract tab in etherscan and in 'mint' put in nonce integer and block 0.

The gas should be between .04 eth and .08 eth.

Mint and enjoy!

AUTO-MINTING

To not miss nonces, it is useful to auto-mint the result.

The mod for mpunks-miner-controller can be used to auto-mint the found nonce. Use http://trust-in.info/latest.txt as the end point to poll.

If there is enough money in the wallet, this will trigger an auto mint.

MINER OPTIONS

        -a 'address' - this is the eth address nonce will be submitted to. use 0 if you do not want to be credited. 
        -x 'cudaDevice' - this is the ordinal number of the device.
        -cController - use a different controller             
        -v - print version and exit
